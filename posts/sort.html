# 希尔排序

  2. i++, 将待排序序列中将间隔为did_idi​的记录分成一组. 组内排序

# 快速排序

时间复杂度: O(nlog2n)O(nlog_2n)O(nlog2​n)

  1. 选取记录K1K_1K1​, 将比K1K_1K1​大的移动到K1K_1K1​前, 比K1K_1K1​小的移动到K1K_1K1​后, 将其分割为 2 个表
  2. 对子表进行快速排序, 直到每个子表长度不超过 1
void QKSort(RecordType r[], int low, int height) {
  if (low < high) {
    pos = QKPass(r, low, high);
    QKSort(r, low, pos-1);
    QKSort(r, pos+1, high);
  }
}

int QKPass(RecordType r[], int low, int height) {
  RecordType x = row[low];
  while (low < high) {
    // high 右 -> 左
    while (low < high && r[high].key >= x.key) {
      high--;
    }
    if (low < high) {
      r[low] = r[high];
      low++;
    }
    // low 左 -> 右
    while (low < high && r[low].key < x.key) {
      low++;
    }
    if (low < high) {
      r[high] = r[low];
      high--;
    }
  }
  r[low] = x;
  return low
}

# 选择排序

每一趟在 n−i+1(i=1,2...n−1)n-i+1(i=1,2...n-1)n−i+1(i=1,2...n−1) 个记录中选取最小的记录作为第 iii 个记录

# 简单选择

时间复杂度: O(n2)O(n^2)O(n2)

# 树形选择(锦标赛排序)

思路: 利用上次比较结果

  1. 将 n 个记录两两比较取较小值, 重复该步骤得到最小值, 同时构建出一个满二叉树(用∞\infin∞补全)
  2. 将最小值用∞\infin∞代替, 并更新其至根节点的路径
  3. 重复上述步骤, 得到排序后的序列
